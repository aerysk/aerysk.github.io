Carpool App

INFO 253B: Back-End Web Architecture Spring 2022

  • Team: Anna Pan, Junyuan Tan, Emily Nguyen, Raguvir Kunani, Smit Dagli
  • A Flask app where users can find people to carpool with to destinations
  • Created with Docker, Redis, RQ and MySQL
  • Login capabilities with JSON web token
  • User and drive information stored in SQL database
  • Utilizes Google Maps API to calculate detour distance and travel time
  • Utilizes Mailgun API to send driver an email when a rider joins/leaves their trip
